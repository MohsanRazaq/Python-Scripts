## 📖 The Philosophy
In cybersecurity, relying solely on third-party tools (like Nmap or Wireshark) can leave blind spots. This repository is dedicated to mastering **Core Python Logic** to build custom security solutions from scratch. 

I avoid high-level libraries (like `ipaddress` or `pandas`) to ensure I understand how to manipulate raw data at the byte and string level.
